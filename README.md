# Gas Booking System

A web app for booking gas cylinders with user and admin roles.

## Features
- User registration/login
- Cylinder booking with payment options
- Booking history
- Admin approval/rejection and notices
- Email notifications

## Setup
1. Clone the repo.
2. Configure Firebase in `firebase-config.js`.
3. Run `firebase serve` for local testing or `firebase deploy` for production.

## Workflow
1. User registers/logs in.
2. Books cylinder, views history.
3. Admin manages requests and notices.
4. Emails are sent for confirmations.

## Execution
Open `index.html` in a browser after setup.
